สวัสดีครับโบ! นี่คือหน้าเว็บแบบมี Sidebar

ลองสังเกตดูว่าตอนเปิดหน้าเว็บมา กล่องเหล่านี้จะค่อยๆ ลอยขึ้นมาทีละอัน นี่คือพลังของ CSS Animation ครับ

ทำไมต้องมี Sidebar?

การมีเมนูทางด้านซ้ายช่วยให้ผู้ใช้งานเข้าถึงส่วนต่างๆ ของเว็บได้ง่ายขึ้นมาก โดยเฉพาะเว็บที่มีเนื้อหาเยอะๆ เหมือนพวก Dashboard ของครูคอมฯ

ลองเอาเมาส์มาชี้ที่กล่องสิ

ผมใส่เอฟเฟกต์ Hover ไว้ด้วย เมื่อโบเอาเมาส์มาวาง กล่องมันจะขยายตัวขึ้นนิดหน่อย (Scale) และเงาจะเข้มขึ้น ทำให้เว็บดูโต้ตอบกับคนใช้ได้ดีขึ้นครับ
